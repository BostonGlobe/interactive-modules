## Install

- `npm install`

## Usage

### Development

- `gulp standalone` for **standalone** graphic.
- `gulp homepage` for **homepage** graphic.

### Production

- `gulp standalone-prod` for **standalone** graphic.
- `gulp homepage-prod` for **homepage** graphic.
